# Getting Started

Welcome to the **Next.js ResMed Dashboard** Storybook.

The purpose of this Storybook installation is to get to know Storybook and how it can be used to develop and document the components of the ResMed Dashboard application. It is not intended to be a full implementation, nor a full and accurate representation of this application.

That said, this Storybook install shows the component library and development environment for the ResMed Dashboard. Browse the sidebar to explore individual components and their documentation.

## Components

- **DisplayJSON** — Renders structured JSON data in a readable format
- **SegmentedToggle** — A segmented control for switching between options
- **Select** — A custom dropdown select component
- **Switch** — A toggle switch component
- **Input** — A styled text input component
