# DisplayJSON

DisplayJSON is a way to display JSON data in a readable format. It is a component that can be used to display JSON data in a structured way, making it easier for users to understand the data.

## Design implementation

To help users get acquainted with the existing UI elements, it is recommended to use check the Figma file to see how the select input is implemented.

### When to use?

When you need to see some JSON data in a readable format, you can use the DisplayJSON component. It is especially useful when you need to display complex JSON data that is difficult to read in its raw form.
